概要

Go Versionsによると、 1.21rc2 のようなGoのバージョンは、Release Candidateであり、リリース候補を示す。リリースされたら 1.21.4 のようになる。ほかのアプリケーションでも見かける表記だが、あまり考えてこなかった。 

1.21 < 1.21rc1 < 1.21rc2 < 1.21.0 < 1.21.1 < 1.21.2

1.21がリリース候補より前、という意味になるのは落とし穴に見える。
