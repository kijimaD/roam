* 概要

Go言語の ~big.Rat~ 型は分数を表すための型である。ゼロ値でのゼロ除算を避けるため、分母は1として扱われている。

#+begin_src go
  import (
    "math/big"
    "fmt"
  )

  func main() {
    var zero big.Rat
    fmt.Printf("%#v\n", zero.Num()) // 分子
    fmt.Printf("%#v\n", zero.Denom()) // 分母

    fmt.Printf("%#v\n", zero)
    // aは分子、bは分母
  }
#+end_src

#+RESULTS:
#+begin_src
0
1
big.Rat{a:big.Int{neg:false, abs:big.nat(nil)}, b:big.Int{neg:false, abs:big.nat(nil)}}
#+end_src
