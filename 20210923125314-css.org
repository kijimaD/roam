#+title: CSS
* 概要
CSSは[[id:9f5b7514-d5e5-4997-81b0-bd453775415c][HTML]]をどのように表示するかを指定する仕様のこと。
* Memo
** テーブルの間隔を指定する
#+begin_export css
table {
  border-spacing: 2em;
  border-collapse: separate;
}
#+end_export
** tbodyに間隔を指定する
~border-spacing~ は使えないので、要素を挿入して間を開けるしかない。
#+begin_export css
tbody::before
{
  content: '';
  display: block;
  height: 1em;
}
#+end_export
