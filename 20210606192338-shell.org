** ディレクトリの全ファイルで実行する
#+begin_src shell
  for file in `\find ./src -name '*.py'`;
  do
  echo $file
  python $file | sed -e s/.*[0-9]m// >> ./docs/query.org
  done
#+end_src

** suspend
コマンドでサスペンドする方法。

Ubuntuのとき。

  sudo pm-suspend
#+end_src

GNU Guixのとき。
#+begin_src shell

#+end_src
** pgrep, pkill
