概要

React Query は外部リクエストに関する状態管理をする。外部リクエストは非同期に行われており、さまざまな状態がある。リクエスト中、リクエスト完了、読み込み中、読み込み完了など。それらを自らで実装する必要なく、効率的にやってくれる。
