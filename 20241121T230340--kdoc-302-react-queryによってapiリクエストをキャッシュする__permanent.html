概要

React QueryはAPIリクエストのキャッシュを行う。同じパラメータの場合はキャッシュから取ってきたりする。また、変更があった要素だけ再レンダリングもできる。

直接axiosを呼び出すと毎回リクエストを飛ばすことになるだろう。リクエストの分遅くなるだけでなく、WEB APIサーバの負荷を高めることになる。
