* 概要

コーディング問題に、数値を桁ごとに処理せよ、というものがある。さっと出てこなかったのでメモしておく。

#+caption: "12345"を、1桁ずつ処理する
#+begin_src go
  import "fmt"
  func main() {
          for _, r := range "12345" {
                  c := int(r - '0')
                  fmt.Println(c)
          }
  }
#+end_src

#+RESULTS:
#+begin_src
1
2
3
4
5
#+end_src

流れ。

1. 入力文字列(数字)をrune、つまりUnicodeコードポイントに分割する
2. 1文字ごとにベースのコードポイント'0'を使って、コードポイントを数値に変換する。数字のコードポイントは連番で並んでいる
3. 得られた数値を使う
