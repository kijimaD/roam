* 概要

正規表現ではマッチするとカーソルが進み、2回同じ文字でマッチしない。正規表現エンジンで共通の挙動なのか、異なる挙動のものがあるかは調べていない。

#+caption: 直感的には、"B8", "8B" がマッチしそうに見える
#+begin_src go
    package main

    import (
            "fmt"
            "regexp"
    )

    func main() {
            found := regexp.MustCompile("B8|8B").FindAllString("B8B", -1)
            fmt.Printf("%#v\n", found)
    }
#+end_src
