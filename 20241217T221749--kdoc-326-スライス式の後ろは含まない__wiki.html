概要

Go言語のスライス式では、後ろのインデックスは含まない。

  • 前index <= value < 後index
package main

import "fmt"

func main() {
        a := []string{"a", "b", "c", "d", "e"}
        fmt.Println("len:", len(a))
        fmt.Println("4:  ", a[4]) // 👈 同じ「4」だが意味は違う
        fmt.Println("0-4:", a[0:4]) // 👈
        fmt.Println("0-5:", a[0:5])
}

len: 5
4:   e
0-4: [a b c d]
0-5: [a b c d e]

関連

なし。
