* 概要

Goで変数の交換が必要なとき、短く書ける。

#+caption: 変数交換
#+begin_src go
  import "fmt"

  func main(){
          a, b := 1, 2
	    a, b = b, a
          fmt.Printf("%d, %d", a, b)
  }
#+end_src

#+begin_src
