概要

「最初と最後を除いて、要素を足したい」というケースは多い。 

"ABC" => "A B C"

みたいなとき。何も考えずにforループで書くと、先頭や末尾にも半角スペースを足してしまったりする。このように真ん中の要素だけ要素を足したい場合は、 strings.Joins を使うと便利である。 

import (
        "strings"
        "fmt"
)
func main() {
        alpha := []string{"this", "is", "hello", "world"}
        fmt.Print(strings.Join(alpha, "★"))
}
