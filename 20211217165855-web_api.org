** Swagger
SwaggerはAPIドキュメントツールである。ymlやjsonを記述して、ブラウザで閲覧できる。
[[id:e04aa1a3-509c-45b2-ac64-53d69c961214][Rails]]でのラッパーライブラリはrswagで、自然に扱うことができる。
