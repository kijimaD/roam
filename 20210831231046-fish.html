fish

概要

fishはShellのひとつ。特に設定せずとも使い勝手のよい補完を使い始めることができる。

Memo

ブラウザで設定

fish_config

でブラウザで設定できる。

パッケージシステム

fishにパッケージシステムomfをインストールする。 

sudo apt install omf
curl -L https://get.oh-my.fish > install
fish install --path=~/.local/share/omf --config=~/.config/omf
